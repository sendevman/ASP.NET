Introduction

In a web application, there are javascript, css, xml... files used by clients. They are generally added web project and published as seperated files. Sometimes, it may needed to package some of these files into an assembly (say, into a class library project (dll)) and distribute as embedded resource in the assembly. ASP.NET Boilerplate provides an ifrastructure to make this easy.
